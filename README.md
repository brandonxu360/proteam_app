# Proteam App

This project is a diet-tracking app primarily focused on calories and macronutrients.

### Collaborators:
- Brandon Xu
- Camden Dowhaniuk
- Abdur Islam
- Aden Athar

### To run locally:
1. [Install Flutter](https://docs.flutter.dev/get-started/install) and set up your environment
2. Clone this repository
3. [Register for a FoodData Central API key](https://fdc.nal.usda.gov/api-key-signup.html) to use in your environment
4. Build an APK to run on a physical device or build using an emulator (ie. android emulator on Android Studio or IOS emulator on Xcode)
